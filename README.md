# 基于爬虫的电商购物推荐平台

## 项目概述

本项目是一个基于网络爬虫技术的电商购物推荐平台，旨在通过爬取商品数据，经过数据清洗和特征工程处理后，使用多种推荐算法为用户推荐商品。

## 功能特点

- **网络爬虫**: 爬取电商网站商品信息
- **数据清洗**: 对原始数据进行清洗和标准化
- **推荐算法**: 支持热门推荐、基于内容的推荐和协同过滤
- **数据存储**: 使用SQLite数据库存储商品和用户数据
- **用户系统**: 支持用户注册、登录和行为记录
- **个性化推荐**: 基于用户行为提供个性化商品推荐
- **公共推荐查看**: 任何人都可以查看系统为特定用户生成的推荐

## 推荐算法说明

### 1. 热门推荐 (Popularity-based)
根据商品的销量或受欢迎程度进行推荐，适用于新用户。

### 2. 基于内容的推荐 (Content-based)
根据商品的特征（如价格、类别等）计算相似度，为用户推荐与已浏览商品相似的商品。

### 3. 协同过滤 (Collaborative Filtering)
基于用户行为数据，找到相似用户喜欢的商品进行推荐。

## 系统功能

### 用户功能
- **用户注册**: 新用户可以注册账户
- **用户登录**: 已注册用户可以登录
- **行为记录**: 系统记录用户点击、浏览商品的行为
- **个性化推荐**: 根据用户行为提供个性化商品推荐

### 推荐功能
- **热门推荐**: 为新用户或无行为数据的用户推荐热门商品
- **协同过滤推荐**: 基于相似用户喜好推荐商品
- **内容推荐**: 基于商品特征推荐相似商品
- **公共推荐查看**: 任何人都可以查看系统为特定用户生成的推荐

### 数据管理
- **数据爬取**: 自动从电商网站爬取商品数据
- **数据清洗**: 清洗和标准化原始数据
- **特征工程**: 提取商品特征用于推荐算法

## API接口

- `/` - 首页，展示热门商品或个性化推荐
- `/user/<int:user_id>` - 展示特定用户的推荐商品（需登录）
- `/public/user/<int:user_id>` - 公共接口，展示特定用户的推荐商品（无需登录）
- `/product/<int:product_id>` - 展示与指定商品相似的商品
- `/all_products` - 展示所有商品
- `/register` - 用户注册页面
- `/login` - 用户登录页面
- `/logout` - 用户登出
- `/api/recommend/<int:user_id>` - API接口，返回用户的推荐商品（JSON格式）
- `/api/record_click/<int:product_id>` - API接口，记录用户点击商品的行为
